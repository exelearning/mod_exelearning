# Checklist: experimento reproducible

- [ ] Hipótesis enunciada (lo que se espera observar)
- [ ] Comando exacto de setup (copy-pasteable)
- [ ] Comando exacto de ejecución
- [ ] Versión de Moodle / PHP / navegador / SO documentadas
- [ ] Commit hash de cada repo implicado
- [ ] Métricas registradas: cuáles, en qué unidades, número de muestras
- [ ] Evidencias guardadas (logs, capturas) referenciadas por ruta
- [ ] Limitaciones explícitas
- [ ] Resultado interpretado contra la hipótesis
- [ ] Reproducible por un tercero sin contacto con el autor
