# SCORM demo fixture

`actividad-evaluable_scorm.zip` — a **SCORM 1.2** package exported from
eXeLearning v4 (same "Actividad evaluable" content as the `.elpx` fixture,
exported in SCORM format). Used as a demo evaluable activity for `mod_scorm`
in the `EXEDEMO` course (`scripts/setup_demo.php` and `blueprint.json`), so the
eXeLearning grading/attempts/completion can be compared against Moodle's native
SCORM module.

- **Source:** provided by the plugin maintainer (ATE), eXeLearning SCORM export.
- **License:** same as the eXeLearning content (Creative Commons
  Attribution-ShareAlike 4.0, per the package `content.xml`).

Demo content only; excluded from the distributed plugin ZIP via `.distignore`.
